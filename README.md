# Analyzing-E-Learning-Platform-Purchases-using-MySQL


## Overview
The analysis was conducted on learner, course, and purchase data from the online learning platform to understand customer purchasing behaviour, revenue performance, and course popularity across categories.

## Key Insights
## 1. Top Revenue-Generating Category
•	Data Analytics generated the highest revenue among all course categories.
•	Courses such as Power BI Masterclass and Advanced SQL contributed significantly to total sales.
## 2. Most Purchased Courses
•	Python for Beginners
•	SEO Fundamentals
•	Advanced SQL
These courses recorded the highest quantities sold, showing strong learner demand.
## 3. Highest Spending Learners
•	Learners from India and the USA showed higher total spending compared to other countries.
•	Some learners purchased courses from multiple categories, indicating cross-domain learning interest.
## 4. Learner Behaviour
•	Several learners purchased more than one course, showing good customer retention and repeat engagement.
•	Multi-category purchases suggest learners are interested in expanding skill sets beyond one specialization.
## 5. Course Coverage
•	All available courses were purchased at least once, indicating healthy course visibility and platform engagement.


## Business Recommendations
Focus Marketing on High-Performing Categories
•	Increase promotions for Data Analytics and Programming courses since they generate the highest revenue and demand.
Bundle Related Courses
•	Create course bundles such as:
1.	Power BI + Advanced SQL
2.	Python + AWS Basics
•	This can increase average purchase value.
Target Repeat Learners
•	Offer loyalty discounts or personalized recommendations to learners who already purchased multiple courses.
Expand Popular Categories
•	Introduce advanced or intermediate-level courses in:
1.	Data Analytics
2.	Programming
•	These categories show strong learner interest.
Improve Global Reach
•	Countries with lower learner participation can be targeted through region-specific campaigns and localized pricing strategies.
## Conclusion
The platform demonstrates strong performance in technical learning categories, especially Data Analytics and Programming. Learners show repeat purchase behaviour and interest in multi-domain learning. Strategic marketing, bundled offerings, and expansion of popular categories can further increase revenue and learner engagement.
